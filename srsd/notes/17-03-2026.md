# Segurança de Redes e Sistemas Distribuídos

## Authentication in a Distributed System

### Unilateral authentication

In unilateral authentication, only one party (either the client or the server) is authenticated. For example, in a typical web application, the server is authenticated to the client using SSL/TLS certificates, but the client may not be authenticated to the server. This can be sufficient for certain applications where the server needs to prove its identity to the client, but it may not be necessary for the client to authenticate itself to the server.
